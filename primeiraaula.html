Adobe DreaWeaver 

O que é e para que serve o Drea Weaver?

É um programa de design web que permite criar, editar e gerenciar sites. Ele combina uma interface visual com um editor de código.

É possível criar sites, aplicativos web e formulários. Exemplo: Criar sites dinâmicos, criar páginas de comércio eletrônico, criar blogs, criar boletins informativos, criar portfólios, criar e-mails em HTML e criar páginas de informações.

Como trabalhamos ou fazemos.

Temos duas opções de usarmos o DreamWeaver, podemos criar os conteúdos dos sites incorporando os elementos por itens nos Menus (comandos), ou criarmos o site programando (TAGs) com códigos em HTML, CSS e outras linguagens de programação Web.
